# Sudoku Web3 Project

A decentralized Sudoku game powered by Ethereum smart contracts.

## Features

- Connect MetaMask wallet
- Play Sudoku
- Submit scores on-chain
- Global leaderboard
- NFT achievement rewards
- Open-source

## Tech Stack

- Solidity
- Hardhat
- React
- Ethers.js
- OpenZeppelin

## Project Structure

```
contracts/
frontend/
scripts/
test/
```

## Installation

```bash
git clone https://github.com/yourusername/sudoku-web3-project.git

cd sudoku-web3-project

npm install
```

## Compile

```bash
npx hardhat compile
```

## Test

```bash
npx hardhat test
```

## Deploy

```bash
npx hardhat run scripts/deploy.js --network sepolia
```
